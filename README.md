# 📝 To-Do List (Python)

A simple command-line To-Do List application built using Python.  
This project allows users to add, view, and remove tasks easily.


# Features
- Add new tasks ✅  
- View all tasks 📋  
- Remove tasks 🗑️  
- Simple and user-friendly interface  


# Technologies Used
- Python (Basic concepts: lists, loops, conditions)


# 📂 How to Run
1. Make sure Python is installed on your system  
2. Download or clone this repository  
3. Open terminal or command prompt  
4. Run the file:

```bash
python todo.py

```
# Example Output

  TO-DO LIST 
1. Add Task
2. View Tasks
3. Remove Task
4. Exit
 
# Purpose
This project is created for practice and to improve basic Python programming skills like lists, loops, and user input handling.
